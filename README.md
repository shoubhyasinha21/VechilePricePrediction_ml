# 🚗 Vehicle Price Prediction System

This is a Machine Learning-powered Web Application built with **Streamlit**. It predicts the market price of vehicles in both **US Dollars (USD)** and **Indian Rupees (INR)** based on historical data.



## 📌 Project Overview
The application uses a **Random Forest Regressor** to estimate vehicle values. It analyzes multiple features such as the manufacturer (make), model, manufacturing year, mileage, fuel type, transmission, and body style to provide an accurate valuation.
